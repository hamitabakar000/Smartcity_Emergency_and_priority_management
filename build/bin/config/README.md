# Dossier de Configuration

Ce dossier contient tous les fichiers de configuration pour le simulateur Smart City Emergency.

## Fichiers disponibles :

### 1. config.json
Configuration principale de la simulation. Contient :
- Paramètres de la fenêtre
- Paramètres de simulation
- Paramètres des véhicules
- Paramètres des feux de circulation
- Positions des bâtiments

### 2. scenarios.json
Scénarios de test prédéfinis pour démonstrations et tests.

### 3. metrics_schema.json
Schéma pour les métriques et statistiques de performance.

### 4. test_config.json
Configuration pour les tests unitaires et d'intégration.

### 5. controls.json
Configuration des contrôles clavier et souris.

### 6. graphics.json
Paramètres graphiques et visuels.

## Comment modifier la configuration :
1. Éditez les fichiers JSON avec un éditeur de texte
2. Redémarrez l'application pour appliquer les changements
3. Certains paramètres peuvent nécessiter une recompilation

## Format JSON :
Tous les fichiers utilisent le format JSON standard. Assurez-vous de :
- Maintenir la syntaxe JSON valide
- Ne pas supprimer les virgules de fin
- Utiliser des guillemets doubles pour les chaînes
